Dover I-2 / I-3 Failure to reach UP High Speed

  • UP fast Solenoid Coil is not energizing.Check for a voltage on the controller terminals for the solenoids. If there is a voltage, then remove one of the terminals and check for a resitance accross the solenoid coil wires. If there is no measureable resitance within 2000 ohms, then replace coil.
  • UP Slow and Fast Solenoid Coils are swapped.
  • Pump FailureCheck for rolling oil in tank while pump is running. If there is very little to no movement of oil, check pump. If there are belts, check for slippage.
  • Pressure Relief OUT too far.Check with pressure gauge.
  • Obstruction in Valve or Jack line.Check jack port with pressure gauge. If pressure at or near what the relief pressure should be, make sure pit valve is open. If pressure is still high, there could be an obstruction cause by muffler.
  • Debris in Solenoid This could cause the solenoids to not open fully. Check UP solenoids for debris.
